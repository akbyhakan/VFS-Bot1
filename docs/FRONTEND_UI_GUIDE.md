- **Background**: Dark Navy (#0f172a)

### Desktop (>768px)
- Sidebar always visible
- Multi-column layouts
- Large statistics cards
- Wide tables

### Tablet (768px)
- Collapsible sidebar
- 2-column layouts
- Medium cards
- Scrollable tables

### Mobile (<768px)
- Hidden sidebar (menu button)
- Single column
- Stacked cards
- Horizontal scroll tables
