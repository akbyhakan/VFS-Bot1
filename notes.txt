The VFS Python Bot is a simple and efficient automation tool designed for the VFSv2023.C3 project. It performs repetitive tasks automatically using Python and integrates tools like ChromeDriver and FFmpeg. This bot helps streamline workflows and improve overall productivity.


Note:
 1. 'VFSv2023.C3' this is the full and latest version updated at 22.12.2023
 2. Complete bot setup using security key and user password
 3. Setup bot all settings properly
 4. Then run bot for appointment booking
 5. Automatically open google chrome webdriver(if not installed then first install latest version of google chrome)
 6. There are some Captcha solution process(Manually, Automatci, Anti-Captcha, TwoCaptcha, NopeCha)
 7. Then wait some seconds and automatically login vfs account
 8. Then check centre for available slots(It's work multiple centre, category, sub-category). If slot not found then wait some seconds and start slot check again
 9. If get any slot then save multiple personal details
10. Then get date and time for appointment
11. Then get service and review details
12. After complete all process goto the payment page with another user selection browser and this page wait for 15mins for payment
13. If any slot booking then sleep 60/seconds and it will start from first for another user
14. All process and info as file and screenshot store into 'logs-data' and 'screenshots' folder

# How to Download and Use FFmpeg

1. Download FFmpeg from the following link:
	https://github.com/BtbN/FFmpeg-Builds/releases/tag/latest
2. Choose the ZIP file based on your operating system and download it
3. After downloading, extract (unzip) the ZIP file
4. Open the extracted folder and navigate to the bin directory
5. Inside the bin folder, you will find the following files:
	ffplay.exe
	ffprobe.exe
	ffmpeg.exe
6. Copy all three(3) files and paste them into your project folder: VFSv2023.C3/


# How to Download and Use ChromeDriver

1. Download ChromeDriver from the following link:
	https://googlechromelabs.github.io/chrome-for-testing/#stable
2. Select the ZIP file that matches your operating system and download it.
3. After downloading, extract (unzip) the ZIP file.
4. Open the extracted folder and locate the chromedriver.exe file.
5. Copy the chromedriver.exe file.
6. Paste it into the following directory inside your project: VFSv2023.C3/webdriver/
